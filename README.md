# YouTube-utilities

Code to manage a collection of videos.
- upload, delete, organize {playlist}, and check metadata for vids on YouTube
- create a (single, scrolable) webpage [i.e. html] for an enumerated set of vids (including a keyframe image).

Please keep in mind that this code reflects what I wanted to do at the time; it will be incomplete. It will not necessarily be a good match to your particular project. Though I did try to keep things reasonably general and generic. Look at the README files for more info. 
Feel free to branch and push.

I apologize for the dearth of documentation. Ideally '--help' will give you the right information. But check the code so that you know what's actually going on. Feel free to comment.
